Messenger-ээр чатлаж буй хэрэглэгчдэд зориулсан. Gemini function calling ашиглана.

🔍 search_properties▼
Юу хийдэг: Supabase properties хүснэгтээс хэрэглэгчийн хүсэлтэд тохирох үл хөдлөх хайна. Шүүлт: төрөл, үнэ (min/max), өрөөний тоо, дүүрэг, хэмжээ. Олдсон үр дүнг форматлаж, зураг үзэх quick reply товчтой хамт буцаана.
Параметрүүд:
typeapartment | house | office | land | commercial
min_priceДоод үнэ (MNT)
max_priceДээд үнэ (MNT)
roomsӨрөөний тоо
districtДүүрэг/байршил (ilike хайлт)
min_size / max_sizeХэмжээ м²
limitҮр дүнгийн тоо (default: 5)
Ажиллах дараалал:
1. properties хүснэгтэд query бүтээх (shop_id, is_active, status=available)
2. Шүүлтүүд нэмэх (type, price, rooms, district, size)
3. Үр дүнг format хийх (нэр, байршил, хэмжээ, үнэ)
4. Quick reply товч үүсгэх (📷 зураг үзэх)
→ executeSearchProperties()

Параметрүүд:

Ажиллах дараалал:
1. ID эсвэл нэрээр property хайх
2. images array шалгах

→ executeShowPropertyImages()
💰 calculate_loan▼
Юу хийдэг: Орон сууцны зээлийн тооцоолол хийнэ. Сарын төлбөр, нийт төлбөр, нийт хүүг тооцоолно. Default хүү: 12.5% (Хаан банк). DB-д хандахгүй, зөвхөн математик тооцоо.
Параметрүүд:
amountЗээлийн дүн (MNT) — required
rateЖилийн хүү % (default: 12.5)
yearsХугацаа жилээр — required
down_paymentУрьдчилгаа (MNT)
Тооцооллын томъёо:
• M = P × [r(1+r)^n] / [(1+r)^n - 1]
• principal = amount - down_payment
• Сарын + Нийт + Хүүгийн дүн харуулна
→ executeCalculateLoan()
📅 schedule_viewing▼
Юу хийдэг: Хэрэглэгч үл хөдлөх үзэхийг хүссэн бол уулзалт товлоно. Lead үүсгэж/шинэчилж, property_viewings хүснэгтэд бичиж, эзэмшигчид push notification илгээнэ. Монгол хэлний огноо/цаг таньдаг ("маргааш", "өглөө").
Параметрүүд:
property_nameБайрны нэр — required
property_idБайрны ID
preferred_dateОгноо (YYYY-MM-DD / "маргааш")
preferred_timeЦаг ("өглөө"=10, "үд"=14, "орой"=17)
customer_phoneУтасны дугаар
Ажиллах дараалал:
1. Property хайх (нэр/ID)
2. Огноо/цаг parse хийх
3. Lead үүсгэх/шинэчлэх (status: viewing_scheduled)
4. property_viewings хүснэгтэд insert
5. Phone хадгалах (байвал)
6. Push notification илгээх
→ executeScheduleViewing()
