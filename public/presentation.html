Developer Guide

Meta Developer App Setup

Live API call хийж page data авах бүрэн заавар:

1

App үүсгэх

developers.facebook.com → My Apps → Create App → "Business" → "Vertmon Hub"

2

Products нэмэх

Messenger + Webhooks + Facebook Login + Instagram API

3

Page холбох + Token авах

Messenger Settings → Add Pages → Generate Page Access Token → .env.local-д хадгалах

4

Webhook тохируулах

Callback URL: https://vertmonhub.vercel.app/api/webhook
Subscribe: messages, messaging_postbacks, feed

5

Live API test

# Page info авах
curl "https://graph.facebook.com/v18.0/me?access_token=TOKEN"

# Мессеж илгээх
curl -X POST "https://graph.facebook.com/v18.0/me/messages?access_token=TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"recipient":{"id":"PSID"},"message":{"text":"Hello!"}}'

# Page insights
curl "https://graph.facebook.com/v18.0/PAGE_ID/insights?metric=page_impressions&access_token=TOKEN"
